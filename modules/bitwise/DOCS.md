# Bitwise Module
**Description:** Handles raw binary logic gates for low-level memory operations.
### Features:
- `gage_bit_and(a, b)`: Binary AND operator.
- `gage_bit_or(a, b)`: Binary OR operator.
- `gage_bit_xor(a, b)`: Binary XOR operator.
### How to Use:
1. Import: `import bitwise`
2. Example: `let result = gage_bit_and(5, 3)`
