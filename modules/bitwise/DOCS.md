# 🧬 Comprehensive Bitwise Module
Low-level binary manipulation masks, logic gates, and bit shift register performance frameworks.

### Complete Feature List:
1. `gage_bitwise_and(a, b)` - Computes bitwise logical AND matching matrix filters.
2. `gage_bitwise_or(a, b)` - Computes bitwise logical OR to combine status flag rows.
3. `gage_bitwise_xor(a, b)` - Computes bitwise logical XOR to analyze data differences.
4. `gage_bitwise_not(a)` - Inverts all individual bit value profiles entirely.
5. `gage_bitwise_left_shift(a, s)` - Steps bits leftwards executing rapid multiplication.
6. `gage_bitwise_right_shift(a, s)` - Steps bits rightwards executing rapid division math.
7. `gage_bitwise_set_bit(a, n)` - Forces target integer pointer offset bit up to 1.
8. `gage_bitwise_clear_bit(a, n)` - Disables an absolute bit track index directly down to 0.
9. `gage_bitwise_toggle_bit(a, n)` - Reverses state of an exact target bit coordinates cell.
10. `gage_bitwise_check_bit(a, n)` - Pulls specific bit indicator values directly for audits.
11. `gage_bitwise_count_set_bits(n)` - Aggregates total active bit populations inside variables.
12. `gage_bitwise_check_parity(n)` - Audits binary data distribution parity parameters.
13. `gage_bitwise_isolate_lowest_bit(n)` - Isolates lowest set operational bit thresholds layout.
14. `gage_bitwise_clear_lowest_bit(n)` - Clears trailing noise factors by evicting lowest bit.
15. `gage_bitwise_is_power_of_two(n)` - Validates perfect geometric balance inside binary grids.
16. `gage_bitwise_generate_mask(n)` - Compiles continuous bit verification criteria fields.
17. `gage_bitwise_rotate_left(n, s)` - Rotates bits looping upper overflow boundaries home.
18. `gage_bitwise_rotate_right(n, s)` - Rotates bits looping falling lower bits to high indexes.
19. `gage_bitwise_binary_to_gray(n)` - Converts binary entries into Gray code tags layouts.
20. `gage_bitwise_gray_to_binary(n)` - Restores encoded Gray tokens back into standard registers.
21. `gage_bitwise_swap_nibbles(b)` - Exchanges upper and lower half data blocks inside bytes.
22. `gage_bitwise_reverse_bytes(n)` - Changes internal word endian layout sequences instantly.
23. `gage_bitwise_get_bit_length(n)` - Quantifies raw space footprint depth before allocation fields.
24. `gage_bitwise_sign_extend(v, b)` - Scales sign flags across packed data fragments fields safely.
25. `gage_bitwise_zero_extend(v, b)` - Cuts off out-of-boundary artifacts past strict bit thresholds.
26. `gage_bitwise_multiply_by_p2(v, p)` - Multiplies values using rapid bit shifts operations.
27. `gage_bitwise_divide_by_p2(v, p)` - Divides values using rapid bit shifts operations.
28. `gage_bitwise_invert_even_bits(n)` - Inverts bit assignments occupying even layout locations.
29. `gage_bitwise_invert_odd_bits(n)` - Inverts bit assignments occupying odd layout locations.
30. `gage_bitwise_mock_checksum(a, b)` - Mixes registers states generating verification checksum numbers.
