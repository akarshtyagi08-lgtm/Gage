# Math Module
**Description:** Advanced mathematical operations for complex calculations.
### Features:
- `gage_pow(base, exp)`: Calculates the power of a base number.
- `gage_sqrt(num)`: Returns the square root of a number.
### How to Use:
1. Import: `import math`
2. Power example: `let squared = gage_pow(5, 2)` (Returns 25)
3. Sqrt example: `let root = gage_sqrt(49)` (Returns 7)
