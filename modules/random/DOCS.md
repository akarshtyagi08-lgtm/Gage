# Random Module
**Description:** Generates random numbers for games, procedural generation, or logic.
### Features:
- `gage_init_random()`: Seeds the random number generator using the current time.
- `gage_random(max)`: Returns a random integer between 0 and `max - 1`.
### How to Use:
1. Import the module: `import random`
2. Initialize it first: `exec "gage_init_random();"`
3. Generate a number: `let x = gage_random(10)`
