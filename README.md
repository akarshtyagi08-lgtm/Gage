# Gage Programming Language

Gage is a lightweight, high-performance custom programming language compiled natively from scratch in C. It features an optimized custom lexical analyzer engineered to parse and execute code tokens seamlessly across mobile and desktop architectures.

---

## Features
*   **Zero Dependencies:** Built entirely using raw C.
*   **Ultra-Lightweight:** Micro-sized binary footprint under 100MB.
*   **Fully Cross-Platform:** Automated native build pipelines running for Android, Windows, Linux, and macOS.
*   **Global Architecture:** Installs globally into your terminal—no path prefixes required.

---
